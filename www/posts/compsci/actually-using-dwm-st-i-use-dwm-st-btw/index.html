Installing dwm and st is really, really easy. Maybe a bit harder, when you don’t have sudo.

The post was really chaotic, so I’m going to rewrite it to be simple. A tiling window manager is a GUI, that tiles windows in a specific way. You can have them:

  • tiled (side by side),
  • floating (just like in Windows),
  • monocle (all of them maximized, stacked).

What is scary is that, that’s totally enough. So there is this code in C that provides just that. Your leader key is the ‘alt’ key (I mean, you can choose any other), and with specific shortcuts you can do just anything without the use of mouse.
