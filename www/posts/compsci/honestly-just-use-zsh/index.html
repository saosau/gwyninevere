Honestly, Just Use Zsh

Wed Dec 21, 2022 · 205 words

Honestly, Bash compared to Zsh was always a painful shell for me. At least after I went Zsh-team.

To set zsh as your main shell, either use chsh or set the environment variable $SHELL in your .profile. It will ask you for a setup, but you don’t need to go through it, all you need is .zshrc.

.zshrc Â¶

Generally, I don’t understand zshrc. The manual is really bloated, so I just wrote these lines and forgot about them:

[[ ${-#*"i"} = $- ]] && return # non-interactive shell
HISTSIZE=10000
SAVEHIST=10000
HISTFILE="$XDG_STATE_HOME/zsh/history"
autoload -U colors
colors
PS1="%B%{$fg[green]%}[%n@%m %{$fg[white]%}%1~"`
   "%{$fg[green]%}]$%b%{$fg[white]%} "
   autoload -U compinit
   zstyle ':completion:*' menu select
   compinit -d "$XDG_CACHE_HOME/zsh/zcompdump-$ZSH_VERSION"
   zmodload zsh/complist
   bindkey -M menuselect 'h' vi-backward-char
   bindkey -M menuselect 'k' vi-up-line-or-history
   bindkey -M menuselect 'l' vi-forward-char
   bindkey -M menuselect 'j' vi-down-line-or-history

Remember that these XDG base directories must exist! (the zsh folder as well)

Now when you don’t know the contents of a directory just tab it out, you can also select it with hjkl keys. It will stay on the line you are on at the moment instead of being moved down. Also, ssh and scp completion and many more, it’s just too comfortable. I really, really recommend adding in syntax highlighting as well.
