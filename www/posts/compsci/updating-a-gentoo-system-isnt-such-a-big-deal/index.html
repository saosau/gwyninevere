So here’s how you could update your gentoo system.

I usually have quiet and ask paremeters in $EMERGE_DEFAULT_OPTS variable in /etc/portage/make.conf. If you set them beforehand, updating the system looks like this:

emerge -uDN @world # with -av --quiet-build=y in EMERGE_DEFAULT_OPTS
emerge -aqvuDN @world

Super. But in reality, especially if you are on bleeding edge side of Gentoo (for example ~amd64), then you are going to build a lot of packages, and you’re going to wait unimaginably long every weekly update for laptops at least.
